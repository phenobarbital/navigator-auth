Client Credentials Flow

Use curl to test:

curl -X POST http://localhost:5000/oauth2/token \
  -H "Content-Type: application/x-www-form-urlencoded" \
  -d "grant_type=client_credentials&client_id=1&client_secret=test_client_secret"

Setup

Ensure you have created the 'test_client_id' in the backend storage (Memory/Redis/DB).
